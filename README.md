# Valora

Valora is a frontend-only e-commerce practice project inspired by Amazon's shopping experience. It uses plain HTML, CSS, and JavaScript with a structure that is simple to learn and easy to scale.

## Current Setup

- `index.html` is the only main page right now
- extra linked pages can be added later when you build them

## Folder Guide

- `components/` stores reusable HTML partials
- `css/` separates base, component, and page styles
- `js/components/` holds reusable UI logic
- `js/pages/` holds page-specific scripts
- `js/services/` is the future-friendly layer for API and storage code
- `data/products.json` acts like mock backend data for now

## How To Run

Because this project uses JavaScript modules and fetches local files, run it with a local server instead of opening the HTML files directly in the browser.

Examples:

- VS Code Live Server
- `python -m http.server`
- any simple static server
